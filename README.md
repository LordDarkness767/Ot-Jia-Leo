# Ot-Jia-Leo

Una web dirigida a personas calvas o con pérdida de cabello, donde puedan informarse, mejorar su imagen y encontrar soluciones prácticas. El objetivo es informar, ayudar y mejorar la autoestima de los usuarios.

<h1>Objectius</h1>
Ofrecer información clara sobre la calvicie
Dar consejos de cuidado personal
Proponer estilos y looks atractivos
Mostrar posibles tratamientos
Crear una pequeña comunidad
<br>
<h2>De que va el projecte </h2>

Aquest projecte consisteix en la creació d’un blog web centrat en persones amb calvície o pèrdua de cabell. Es tracta d’un blog informatiu i d’estil de vida, on es comparteixen continguts relacionats amb el cuidat personal, consells estètics i informació útil per millorar la imatge i la confiança.

<h2>Tecnologies utilitzades</h2>

Per al desenvolupament d’aquesta web s’han utilitzat tecnologies bàsiques del desenvolupament web, centrades en la creació d’interfícies atractives i funcionals.

🌐 Llenguatges de programació
HTML (HyperText Markup Language)
S’utilitza per estructurar el contingut de la web. Permet definir elements com títols, paràgrafs, imatges o enllaços.
CSS (Cascading Style Sheets)
S’encarrega del disseny i l’estètica de la pàgina. Amb CSS es poden aplicar colors, tipografies, espais i distribució dels elements.
JavaScript (JS)
Permet afegir interactivitat a la web, com ara botons dinàmics, validació de formularis o efectes visuals.
🛠️ Eines utilitzades
Editor de codi (com Visual Studio Code) per escriure i organitzar el codi
Navegador web (Chrome, Edge, etc.) per provar i visualitzar la web
<br>


<h2>Instalacio o us</h2>

A continuació s’expliquen els passos per descarregar i executar el projecte en local, així com la seva publicació en línia.
<br>
💻 Execució en local (amb Live Server)
Descarregar el projecte
Descarrega els fitxers del projecte en format .zip o clona el repositori (si està a GitHub).
Descomprimeix la carpeta al teu ordinador.
Obrir el projecte
Obre la carpeta del projecte amb un editor de codi com Visual Studio Code.
Instal·lar l’extensió Live Server
Ves a la secció d’extensions de Visual Studio Code.
Busca “Live Server” i instal·la-la.
Executar el projecte
Fes clic dret sobre el fitxer index.html.
Selecciona l’opció “Open with Live Server”.
El navegador s’obrirà automàticament amb la web en funcionament.
<br>
🌐 Publicació amb domini (opcional)
Pujar el projecte a una plataforma
Pots utilitzar serveis com GitHub Pages, Netlify o Vercel.
Puja els fitxers del projecte.
Configurar la publicació
Selecciona el fitxer index.html com a pàgina principal.
La plataforma generarà una URL pública.
Afegir un domini personalitzat (opcional)
Compra un domini (per exemple a GoDaddy o Namecheap).
Configura els DNS perquè apuntin a la teva web (segons la plataforma utilitzada).


<h2>Estructura del projecte </h2>

El projecte està organitzat en diferents carpetes per separar els fitxers segons la seva funció:

OT-JIA-LEO/
│
├── index.html → Pàgina principal del blog
├── README.md → Documentació del projecte
│
├── CSS/ → Estils de la web
│ ├── calvo.css
│ ├── Calvo2.css
│ ├── leo.css
│ ├── ot.css
│ └── responsive.css
│
├── html/ → Altres pàgines del blog
│ ├── calvos.html
│ └── gatos.html
│
└── img/ → Imatges utilitzades a la web


🧾 Explicació ràpida
index.html → és el punt d’entrada de la web
CSS/ → conté tots els estils per donar disseny a la pàgina
html/ → pàgines secundàries del blog
img/ → recursos visuals (imatges)
README.md → informació sobre el projecte
